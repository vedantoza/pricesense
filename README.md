# DataCo Discount Optimization

A focused SvelteKit application for fitting quadratic profit curves by region and department and recommending a profit-maximizing discount.

## Local development

```bash
npm install
npm run check
npm run dev
```

## Cloudflare Pages

Use these build settings:

- Build command: `npm run build`
- Build output directory: `build`
- Root directory: `/`
- Production branch: `main`

No environment variables are required.

## GitHub Pages

For the repository path `/pricesense`, build with:

```bash
npm run build:github
```
